ভূমিকা

CK Book প্রজেক্টে আপনাকে স্বাগতম!

আমরা সবাই একসাথে শিখছি! এই ওয়েবসাইটটি আমাদের নিজেদের শেখা সহজ করতে এবং বন্ধুদের সাহায্য করতে তৈরি করা হয়েছে। আপনি যদি আপনার তৈরি করা কোনো চিট শিট বা হ্যান্ড নোট সবার সাথে শেয়ার করতে চান, তবে গিটের (Git) মাধ্যমে এই প্রজেক্টে অবদান (Contribute) রাখতে পারেন।

অবদান রাখার সাধারণ নিয়মাবলী:

  • স্বচ্ছ কোডিং: আমরা যেহেতু ক্লাসে HTML শিখেছি, তাই আমরা শুধু HTML ফাইল এডিট করেই এই কাজটা করব। কোনো জটিল CSS/JS এডিট করার প্রয়োজন নেই।
  • ইমেজ কোয়ালিটি: চিট শিটের ছবিটি যাতে পরিষ্কার এবং পড়ার উপযোগী হয় তা নিশ্চিত করুন।
  • সঠিক ক্রেডিট: চিট শিটের নিচে আপনার নাম ও ফেসবুক প্রোফাইলের লিঙ্ক দিতে ভুলবেন না যাতে সবাই জানতে পারে কাজটি আপনার!

সাহায্য প্রয়োজন?

বাম পাশের মেনু থেকে ধাপগুলো একে একে অনুসরণ করুন। এখানে খুব সহজ ভাষায় বিস্তারিত গাইডলাইন দেওয়া আছে।

ধাপ ০১

প্রজেক্ট ফোর্ক (Fork) ও ক্লোন (Clone) করা

প্রজেক্টের সোর্স কোড আপনার নিজের গিটহাব অ্যাকাউন্টে কপি করে এনে কাজ শুরু করার জন্য নিচের ধাপগুলো অনুসরণ করুন:

১
GitHub রিপোজিটরিতে যান:

প্রথমে ব্রাউজারে আমাদের মূল প্রোজেক্ট রিপোজিটরি লিঙ্কে প্রবেশ করুন: github.com/RusulAzom/ck-book

২
প্রজেক্টটি ফোর্ক (Fork) করুন:

পেজের ওপরের ডান কোণায় থাকা Fork বোতামে ক্লিক করুন। এরপর Create fork এ ক্লিক করুন। এর ফলে পুরো প্রজেক্টটি আপনার গিটহাব অ্যাকাউন্টে কপি হয়ে যাবে।

৩
ফোর্ক করা রিপোজিটরির URL কপি করুন:

ফোর্ক হওয়ার পর ব্রাউজারের অ্যাড্রেস বার থেকে আপনার রিপোজিটরির লিঙ্কটি কপি করুন (যেমন: https://github.com/YOUR_USERNAME/ck-book)।

৪
আপনার কম্পিউটারে ক্লোন (Clone) করুন:

আপনার কম্পিউটারে কমান্ড প্রম্পট/টার্মিনাল অথবা VS Code-এর টার্মিনাল ওপেন করুন এবং কপি করা লিঙ্কটি দিয়ে ক্লোন করুন:

Terminal 
git clone https://github.com/YOUR_USERNAME/ck-book.git

*বিঃদ্রঃ YOUR_USERNAME এর জায়গায় আপনার নিজস্ব গিটহাব ইউজারনেম ব্যবহার করুন।

ধাপ ০২

আপনার চিট শিটের ছবি ফোল্ডারে যুক্ত করা

ক্লোন করা প্রজেক্ট ফোল্ডারটি VS Code-এ ওপেন করুন এবং আপনার তৈরি করা চিট শিটের ইমেজ ফাইলটি সঠিক জায়গায় পেস্ট করুন:

  • আপনার চিট শিটের ইমেজ ফাইলের একটি অর্থবহ নাম দিন (যেমন: html-tags-cheat-sheet.jpg বা css-flexbox-note.jpg)।
  • ছবিটি নিচের নিয়ম অনুযায়ী সঠিক ক্যাটাগরির ফোল্ডারে পেস্ট করুন:
    • HTML চিট শিট হলে: imgs/cheatsheet/module 1/ ফোল্ডারে।
    • Git & GitHub চিট শিট হলে: imgs/cheatsheet/module 2/ ফোল্ডারে।
    • CSS চিট শিট হলে: imgs/cheatsheet/module 3/ ফোল্ডারে (ফোল্ডার না থাকলে তৈরি করে নিন)।
    • JavaScript চিট শিট হলে: imgs/cheatsheet/module 4/ ফোল্ডারে (ফোল্ডার না থাকলে তৈরি করে নিন)।

ইমেজ ফরম্যাট:

চেষ্টা করবেন চিট শিটের ছবিটি .jpg বা .png ফরম্যাটে যুক্ত করতে।

ধাপ ০৩

HTML কোড আপডেট করা

আপনার কার্ডটি চিট শিটের গ্যালারিতে দেখানোর জন্য আপনাকে HTML ফাইলে একটি কোড ব্লক যুক্ত করতে হবে। আমরা যেহেতু ক্লাসে HTML শিখেছি, তাই এটি খুবই সহজ হবে!

কীভাবে করবেন?

  • VS Code-এ cheatsheet.html ফাইলটি ওপেন করুন।
  • যে মডিউলে ছবি যোগ করেছেন, সেই মডিউলের সেকশনটি খুঁজুন (যেমন: HTML এর জন্য <section id="sec-html-cheatsheet">)।
  • সেই সেকশনের ভেতরে থাকা <div class="cheatsheet-gallery"> ট্যাগটি খুঁজুন।
  • নিচের কোডটি কপি করে গ্যালারি ডিভের ভেতরে পেস্ট করে দিন:
HTML Template 
<!-- নতুন চিট শিট কার্ড শুরু -->
<div class="gallery-card">
    <div class="gallery-image-container" onclick="openLightbox('imgs/cheatsheet/module X/your_image.jpg', 'আপনার চিট শিটের নাম')">
        <img src="imgs/cheatsheet/module X/your_image.jpg" alt="আপনার চিট শিটের নাম">
        <div class="gallery-image-overlay">
            <div class="preview-icon-wrapper">
                <i class="fa-solid fa-magnifying-glass-plus"></i>
            </div>
        </div>
    </div>
    <div class="gallery-info">
        <h3 class="gallery-title">চিট শিটের নাম (যেমন: HTML Semantic Tags Note)</h3>
        <div class="gallery-author">
            <span>লেখক:</span>
            <a href="আপনার_ফেসবুক_বা_প্রোফাইল_লিঙ্ক" target="_blank" class="author-link">
                <i class="fa-brands fa-facebook"></i> আপনার নাম
            </a>
        </div>
        <a href="imgs/cheatsheet/module X/your_image.jpg" download="your_image.jpg" class="gallery-download-btn">
            <i class="fa-solid fa-download"></i> ডাউনলোড করুন
        </a>
    </div>
</div>
<!-- নতুন চিট শিট কার্ড শেষ -->

কোড এডিট করার নির্দেশাবলী:

১. module X/your_image.jpg এর জায়গায় আপনার ছবির ফোল্ডার ও সঠিক ফাইলের নাম লিখুন।
২. আপনার চিট শিটের নাম ও চিট শিটের শিরোনাম এ আপনার প্রজেক্টের নাম লিখুন।
৩. আপনার_ফেসবুক_বা_প্রোফাইল_লিঙ্ক এ আপনার ফেসবুক বা যেকোনো সোশ্যাল মিডিয়া প্রোফাইলের লিঙ্ক দিন।
৪. আপনার নাম এর জায়গায় আপনার নাম লিখে দিন।

ধাপ ০৪

পরিবর্তন GitHub-এ আপলোড ও পুল রিকোয়েস্ট (PR) পাঠানো

কোড এডিট এবং ইমেজ অ্যাড করার পর আপনার করা কাজগুলো আমাদের মূল রিপোজিটরিতে জমা দেওয়ার নিয়ম:

১
পরিবর্তনগুলো যুক্ত (Stage) ও কমিট (Commit) করুন:

VS Code টার্মিনালে এই কমান্ডগুলো একে একে রান করুন:

Terminal 
git add .
git commit -m "feat: added X cheatsheet by yourname"
২
কোড গিটহাবে পুশ (Push) করুন:

আপনার ফোর্ক করা রিমোট অ্যাকাউন্টে কোড পাঠাতে নিচের কমান্ডটি দিন:

Terminal 
git push origin main
৩
পুল রিকোয়েস্ট (Pull Request) পাঠান:

এখন ব্রাউজারে আপনার গিটহাব অ্যাকাউন্টে ফোর্ক করা ck-book রিপোজিটরি পেজে যান। সেখানে একটি Compare & pull request বাটন দেখতে পাবেন, ওটাতে ক্লিক করুন। এরপর একটি সুন্দর টাইটেল দিয়ে Create pull request বোতামে ক্লিক করুন।

অভিনন্দন!

আপনার পাঠানো পুল রিকোয়েস্টটি আমাদের কাছে চলে আসবে। সফল কন্ট্রিবিউশনের জন্য অগ্রিম ধন্যবাদ!

রিভিউ শিডিউল

মডারেটর রিভিউ ও ওয়েবসাইট আপডেট শিডিউল

আপনার পাঠানো কন্ট্রিবিউশনটি (Pull Request) আমাদের ওয়েবসাইট সিস্টেমে সরাসরি যুক্ত হওয়ার আগে যাচাই করা হবে।

দৈনিক আপডেট সময়:

ওয়েবসাইট মডারেটর (Rusul Azom Sumon) প্রতিদিন রাত ১২:০০ টা (12:00 AM)-এ সমস্ত পুল রিকোয়েস্ট এবং কোড পরিবর্তনগুলো যাচাই (Review) করবেন।

যদি আপনার কোড এবং যুক্ত করা চিট শিটের ইমেজটি সঠিক থাকে, তবে সেটি মূল প্রজেক্টে যুক্ত (Merge) করে নেওয়া হবে এবং ওয়েবসাইটটি লাইভ আপডেট হয়ে যাবে।
